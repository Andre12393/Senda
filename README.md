# Senda
Descripcion del proyecto: La descripcion mas adecuada de la aplicasion seria una app que te ayuda a administrar tu promedio estudiantil al sumar la unidasdes funciona para poder llevar acabo una mejor administracion de las calificasiones de los estudiante y les permite poder ir calculando cuantos puntos lleva y poder manntener un promedio de forma mas facil esta aplicasion estaria destinada para uso en telefonos.


Odjetivo: El odjetivo de la aplicasion es perrmitirle alos usuarios una herramienta que les ayudadia a poder administrar y llevar un registro claro de los promedios de las materias esto ayuda para mantener el promedio de una forma mas controlada al estarlo monitoriando constante mente.


Pasos para ejecutarl: Los pasos para ejecutarlo son faciles y sencillos , iniciar o crear una cuenta esto para poder tener bien administrados tus distintas materias , despues encontrara un boton para agregar maria al precionarlo y agregar el nombre de tu materia ya podras agergar las calificasiones de la unidad 1,2 y3 y con eso ya estaria tu promedio de esa materia le das a guardar y ahora podras agregar y usar la aplicasion a voluntad .
